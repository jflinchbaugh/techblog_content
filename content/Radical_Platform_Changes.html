I'm accustomed to being the person calling to ripping out the JVM or the application server and replacing it with the latest and greatest versions, but I tried to be a little more conservative and less crazy at my current work place.

I don't want to be just a programmer, I want to be an architect too. Hmm. Oh well, maybe next time.
