title=Java ME at the Harrisburg JUG date=2007-03-16 15:46:29.116 type=post status=published tags=Mobile, Computers, Java ~~~~~~

I gave a quick little talk at last night's JUG meeting on porting game code to run in Java ME. For demonstration, I ported a little BreakOut game from the previous meeting.

I think it went well enough, but I can tell that people still don't grasp the potential of running software on your phone. There are still too many people just thinking, "I have a phone to make phone calls.".

I was hoping to do more presentations on Java ME, but I guess I won't yet. Maybe I'll keep doing these short talks to keep it in people's minds.
