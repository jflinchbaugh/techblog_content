title=Nokia Firmware Upgrade date=2006-08-25 15:16:16.641 type=post status=published tags=Toys, Mobile, Technology ~~~~~~

Nokia's offering free firmware updates for their phones. Reportedly, this is the first carrier to do this. Maybe I'll need to look at Nokia phones next time I'm shopping. I'd love to have a clear option for upgrades, instead of these sketchy schemes we have now for updating my SonyEricsson: mail-in-your-phone, or the buy credits, run this software, and hope it doesn't destroy your phone, and still not really have any clear guarantees as to what it'll accomplish for you.

I'm afraid to try a flash/unlock service for my phone, because I don't want to break it, and if I can't get the Java security completely unlocked, I don't want to lose that one set of certificates that does allow some software to access priveleged services.
